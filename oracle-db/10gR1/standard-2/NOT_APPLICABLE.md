# NOT APPLICABLE

`Standard Edition 2` (SE2) is **not applicable** to Oracle Database 10g Release 1.

No install/lifecycle/golden-image/rpm tooling is generated for this
combination, by design, to avoid producing invalid scripts.

If Oracle's official documentation for your exact release contradicts this,
update `../metadata.yml` and `../../../docs/oracle-editions-matrix.md`
first, then add the tooling.
